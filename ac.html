What is Assassin's Creed: Al-Andalus?

AC: Al-Andalus is concept art for a fictional game set in the Assassin's Creed universe in Andalusia at the year 1487.
This was a project for university class and it was the first time I used a pen tablet to draw ilustrations.
Also, I did some 3D modeling and animation, and exported it to Unity to test if it was game ready. You can see the final product in the links below.
